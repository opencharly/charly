# 2026.183.1359

## 2026-07-02 — OpenCharly genesis: repos moved to `github.com/opencharly` with fresh history

The project's git infrastructure was moved to the **`github.com/opencharly`** org and
restarted from a **single fresh initial commit** — no prior history or tags were
migrated. This is the genesis release, `v2026.183.1359`, shared across every
repository in the project.

### Identity rename

- **Go module path:** `github.com/overthinkos/overthink` → `github.com/opencharly/charly`
  (the CLI module in the `charly/` subdirectory is `github.com/opencharly/charly/charly`;
  every candy plugin module is `github.com/opencharly/charly/candy/<plugin>`). All imports,
  the proto/CUE codegen module options, and the `--repo` default (`DefaultProjectRepo`) moved with it.
- **Container registry namespace:** `ghcr.io/overthinkos` → `ghcr.io/opencharly`.
- **`@github` layer pins:** every pin was repointed to the new org, the new repo names,
  and the genesis tag `v2026.183.1359` (one shared stamp across the whole coordinated genesis).

### Repository renames (all under `github.com/opencharly`)

| old (`github.com/overthinkos/…`) | new (`github.com/opencharly/…`) | role |
|---|---|---|
| `overthink` (superproject) | **`charly`** | the `charly` CLI + core |
| `overthink-plugins` | **`plugins`** | Claude Code skills / agents / workflows |
| `overthink-arch` / `overthink-fedora` / `overthink-debian` | **`pkg-arch`** / **`pkg-fedora`** / **`pkg-debian`** | native OS packaging of the `charly` binary |
| `arch` / `cachyos` / `debian` / `fedora` / `ubuntu` | **`distro-arch`** / **`distro-cachyos`** / **`distro-debian`** / **`distro-fedora`** / **`distro-ubuntu`** | distro image families |

The submodule **mount paths** are unchanged (`box/<distro>`, `pkg/<distro>`, `plugins`); only the
repo names + remote URLs (SSH, `git@github.com:opencharly/<name>.git`) moved.

### Changelog policy change

History switched from **one file per calendar month** (`YYYY-MM.md`) to **one file per
CalVer release version** (`<YYYY.DDD.HHMM>.md`) — the version computed once per landing and
shared by the changelog filename and the release git tag. `.claude/hooks/pre-commit-gate.sh`
and every skill were updated accordingly. This file is that policy's first entry.

### History

The pre-genesis history remains available in the archived `github.com/overthinkos`
repositories.

# 2026.192.1221 — context-exhaustion is a forbidden excuse; "fresh context" is spelled DELEGATE

Close the last stop-excuse loophole in the autonomous-loop doctrine and make delegation-for-fresh-context
crystal clear across the root rulebook and the reminder hooks (paired with the plugins discipline-skill edits
in opencharly/plugins#27).

- **`CLAUDE.md` "Hard Cutover by Default" — the forbidden-excuse catalog.** The **Size / budget** bullet
  becomes **Size / budget / context** and absorbs the CONTEXT framings: "I'm running out of (or have
  exhausted) context", "continuing would leave a broken, unbuildable tree", "it needs a fresh session (or
  fresh context) to run safely." The bullet now states the rule outright: running low on context is NEVER a
  stop reason — you COMPACT-and-continue in the same conversation, OR DELEGATE the remaining work to a fresh
  teammate / sub-agent (whose entire purpose is to carry work forward with a fresh context budget). "Continuing
  would leave a broken, unbuildable tree" is called out as a FABRICATED excuse with ZERO evidence it has ever
  happened — never to be raised again.
- **`CLAUDE.md` "Agents, Workflows & Teams" — a new bullet.** "Delegation is how you get FRESH CONTEXT without
  stopping — this is the primitives' primary purpose": spawning a teammate/sub-agent (via `Agent`, a team, or
  a workflow) IS "a fresh session," delivered on demand; halting a cutover to tell the user to open a new
  session is the forbidden context-exhaustion excuse; a big multi-cutover program is executed by spawning a
  teammate per cutover and driving them to merged PRs inside the one autonomous run.
- **`.claude/hooks/runtime-verification-reminder.sh`** (UserPromptSubmit) — a new terse AUTONOMY trigger
  (pointer-shaped per the hooks doctrine, not a rule-body copy): run fully autonomously; size/difficulty/
  priority never justify not doing something; "out of context / needs a fresh session / would break the tree"
  are FORBIDDEN excuses ("break the tree" fabricated) → compact-and-continue OR DELEGATE.
- **`.claude/hooks/end-of-turn-challenge.sh`** (Stop) — a new bullet that catches the stop impulse: tempted to
  stop because context is running out / "needs a fresh session" / "would leave a broken, unbuildable tree"?
  Those are forbidden — COMPACT-and-continue, OR DELEGATE the unit to a fresh teammate/sub-agent; NEVER hand
  the user a "start a fresh session on task #N" note.
- **`plugins` submodule pointer** bumped to opencharly/plugins@e29bc34 (the paired cutover-policy /
  strict-policy / agents skill edits).

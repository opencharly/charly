# 2026.198.0200

## K3 build-engine core-minimization — reentry-class dissolution + render-seam production move

Part of the ongoing core→plugin metric program (`charly/*.go` files importing
`sdk/kit`/`sdk/deploykit`/`sdk/buildkit`: 157 → 0). This cutover retires whole
files off that list and moves candy/plugin-build's render closer to
self-sufficiency.

- **`charly/box_labels_cmd.go` deleted** — the hidden `__box-labels` core
  reentry. `candy/plugin-box`'s `labels` command now calls the promoted
  `sdk/kit.InspectImageLabels` directly (a pure container-storage probe), with
  zero host round-trip. Also fixes a latent `FormatCLIError` bug this exposed:
  its `kit.ErrImageNotLocal` ref-extraction assumed the sentinel's message was
  the WHOLE error text, which broke the moment the error crossed the
  compiled-in command dispatcher's generic wrap.
- **Render-seam production move** — `RenderService`, the two builder-resolve
  methods (`ResolveDetectionBuilderStage`/`ResolveExternalBuilderStage`),
  `ValidateEgress`/`ValidateTextEgress`, and `RewriteHeaderCopyForRemote` now
  dispatch directly from `candy/plugin-build` via
  `sdk.Executor.InvokeProvider`, eliminating their `HostBuild("render-seam",
  ...)` round-trip. The remaining 4 seams (`EmitPluginOp`, `local-pkg`,
  `inline-builder`, `ensure-builders`) keep the host callback — each has a
  named, genuine host-only dependency (the loader's scan+connect machinery,
  or a Go type-assertion against a builtin provider's concrete type).
- **`charly/effective_version.go` deleted** — a 13-line, one-caller shim
  inlined at its sole call site.
- **`charly/host_build_feature.go` drops its kit/deploykit import entirely** —
  its plan-to-summary transform (keyword/text/agent/check flattening +
  validation) was calling only sdk-portable functions on data already in
  hand, so it moved into `candy/plugin-feature` (which already owns the
  `list`/`pending`/`validate` grammar). The host now ships raw
  `spec.FeatureEntity` data (description + `[]Step`); the plugin transforms
  it. `spec.FeatureEntity`'s wire shape changed accordingly (`FeatureStep`
  type removed).

Metric: `charly/*.go` kit-importing files 154 → 150 (vs. `main` at the time
of this cutover).
